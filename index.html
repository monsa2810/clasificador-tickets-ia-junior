🤖 Clasificador de Tickets con IA

Machine Learning aplicado a Soporte Técnico

⚠️ Nota Importante: Este es un proyecto de Python que requiere ejecución local. GitHub Pages solo muestra información del proyecto. Para usar el clasificador, sigue las instrucciones abajo.

📋 Descripción del Proyecto

Sistema de machine learning básico que clasifica automáticamente tickets de soporte técnico por prioridad (Alta/Media/Baja) basado en el texto de la descripción.

Objetivo: Demostrar aplicación práctica de IA/ML en problemas reales de TI.

✨ Funcionalidades

🎯 Clasificación Automática

Analiza el texto de un ticket y predice su prioridad con 85%+ de precisión.

📊 Modelo Entrenado

Usa scikit-learn con TF-IDF Vectorizer y Naive Bayes para clasificación de texto.

🖥️ Interfaz Web

Aplicación Flask para probar el clasificador de forma interactiva.

💻 Tecnologías

Python 3.x scikit-learn Flask NumPy TF-IDF Naive Bayes

📚 Ejemplos de Clasificación

Entrada: "Servidor caído, nadie puede trabajar"
Predicción: 🔴 PRIORIDAD ALTA
Entrada: "Impresora no conecta a la red"
Predicción: 🟡 PRIORIDAD MEDIA
Entrada: "Consulta sobre uso de sistema"
Predicción: 🟢 PRIORIDAD BAJA

🚀 Cómo Ejecutar Localmente

Paso 1: Clona el repositorio

git clone https://github.com/monsa2810/clasificador-tickets-ia-junior.git
cd clasificador-tickets-ia-junior

Paso 2: Instala dependencias

pip install -r requirements.txt

Paso 3: Entrena el modelo

python train_model.py

Paso 4: Ejecuta la aplicación

python app.py

Paso 5: Abre tu navegador en

http://localhost:5000
